**Lucky Spin** là ứng dụng vòng quay may mắn tương tác, được xây dựng hoàn toàn bằng công nghệ Web (HTML/JS), cho phép bạn tạo ra các mini-game quay số đơn giản, nhanh chóng[cite: 15].

---

## ✨ Tính năng
*   **Giao diện trực quan**: Vòng quay chuyển động mượt mà.
*   **Tùy biến dễ dàng**: Thay đổi các phần thưởng/nội dung ngay trong mã nguồn.
*   **Không cần cài đặt**: Chỉ cần mở trình duyệt để bắt đầu trải nghiệm.

## 🚀 Cách sử dụng
1. Tải toàn bộ repository về máy.
2. Mở tệp `index.html` bằng bất kỳ trình duyệt web nào (Chrome, Edge, Firefox).
3. Bắt đầu quay để thử vận may của ní!

## 🛠️ Hướng dẫn phát triển
*   **Chỉnh sửa nội dung**: Ní mở tệp `index.html`, tìm đến đoạn mã JavaScript (thường nằm trong thẻ `<script>`) để thay đổi các giá trị trên vòng quay[cite: 15].
*   **Tùy biến giao diện**: Ní có thể điều chỉnh kiểu dáng trực tiếp trong thẻ `<style>` hoặc CSS nhúng[cite: 15].
